🍎 Built-in macOS Tool

Standardized screen reading for Mac.

VoiceOver is the powerful, built-in screen reader for macOS. Unlike third-party tools, it is deeply integrated into the operating system and uses a unique "Interaction" model. Mastering these six essential commands will allow you to navigate any website or application effectively.

1. Launch and Close

VoiceOver is built into every Mac. You can toggle it at any moment with a single shortcut.

  • To Launch or Exit: Press Command + F5.
  • On Laptops: You may need to hold the Fn key as well: Fn + Command + F5.

2. Mute (Silence)

If VoiceOver is reading a long passage and you want to silence the speech instantly without turning off the tool:

  • To Silence: Tap the Control key once. This stops the current speech buffer immediately.

3. Screen Curtain

Screen Curtain turns off your display for privacy while keeping VoiceOver fully functional. This is ideal for working in public spaces.

  • To Toggle: Press VO + Shift + F11.
  • (Note: If using a laptop, you may need to add the Fn key).

4. Navigate by Headers

VoiceOver uses "Quick Nav" or specific VO commands to jump between headings, which represent the outline of the page.

  • Next Header VO + Command + H
  • Previous Header VO + Command + Shift + H

5. Navigate by Links

Jump directly to interactive links to find your destination faster.

  • Next Link VO + Command + L
  • Previous Link VO + Command + Shift + L

6. Navigate Form Fields

Move between text fields, buttons, and other form controls.

  • Next Form Control VO + Command + J
  • Previous Form Control VO + Command + Shift + J
  • Interact with Control VO + Shift + Down Arrow

SimpleAccess Insight: The "VO" Keys

Almost all VoiceOver shortcuts require the VO Modifier. By default, this is the combination of Control + Option pressed together.

Testing Tip: Use the Web Rotor (VO + U) to see a list of all headers, links, and form fields on this page. It's the fastest way to get a bird's-eye view of a website's structure!
